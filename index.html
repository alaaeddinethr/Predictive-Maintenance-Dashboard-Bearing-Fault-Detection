Context

Why bearing faults matter

Bearing failures account for 40–50% of all rotating machine breakdowns, causing unplanned downtime worth billions annually in industrial settings.

Classical condition monitoring relies on fixed thresholds — a technician checks vibration once a week. By the time a fault is detected, it's already critical. This project moves the detection upstream: continuous signal analysis identifies defects at the seed stage, before propagation.

The approach combines classical signal processing (FFT, Hilbert transform, envelope analysis) with modern ML, producing a pipeline that is both physically interpretable and highly accurate.

XGBoost PINN FFT Hilbert Streamlit PyTorch CWRU Dataset SHAP

🏆 Origin

This project extends work done at the MTI Hackathon 2025 (3rd place nationally out of dozens of teams), where a Physics-Informed Neural Network was developed for railway bearing degradation modeling. This repository deepens the approach into a full production-grade pipeline.

📡 Dataset

Uses the CWRU Bearing Dataset from Case Western Reserve University — the most widely cited public benchmark in bearing diagnostics literature. 12kHz drive-end accelerometer, 4 fault types, 3 severity levels.

Signal Processing

52-feature extraction pipeline

52
handcrafted features across 3 domains — all grounded in vibration analysis theory

Features are not arbitrary — each one captures a specific physical phenomenon known to correlate with bearing degradation. Kurtosis detects impulsive shocks. Envelope analysis demodulates amplitude-modulated fault signatures. BPFI/BPFO energy tracks power at theoretically predicted fault frequencies.

TD

Time Domain

Statistical moments and waveform descriptors computed directly on the raw signal window.

  • RMS, Peak, Peak-to-Peak
  • Kurtosis, Skewness
  • Crest factor, Shape factor
  • Impulse factor, Clearance factor
  • Zero crossing rate, Energy
  • L2 norm, Mean absolute
FFT

Frequency Domain

FFT spectrum analysis with fault-frequency targeting using bearing geometry formulas.

  • Spectral centroid, bandwidth
  • Spectral rolloff (85%), flatness
  • Band power (0–500, 500–2k, 2–4k, 4–6k Hz)
  • BPFI energy (harmonics 1-3)
  • BPFO energy (harmonics 1-3)
  • BSF, FTF energy, THD
ENV

Envelope Analysis

Hilbert transform demodulation — the gold standard for bearing fault detection in industry.

  • Envelope RMS, Kurtosis, Skewness
  • Envelope crest factor
  • BPFI/BPFO at envelope spectrum
  • Instantaneous frequency (mean, std)
  • Envelope spectral entropy
  • Modulation index

Usage

Get started in 3 commands

01 — INSTALL

Clone & setup

# Clone the repo
git clone https://github.com/
  alaaeddinethr/
  bearing-fault-detection
cd bearing-fault-detection

# Install dependencies
pip install -r requirements.txt
02 — DATA & TRAIN

Download data & train

# Download CWRU dataset
python src/data/download_data.py --process

# Train all models
python src/models/train.py \
  --model all \
  --save

# Run tests
pytest tests/ -v

Launch the dashboard
